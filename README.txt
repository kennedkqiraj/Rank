Project Overview
This project analyzes the 2024 Fortune 1000 Companies alongside ESG (Environmental, Social, and Governance) ratings. It includes data processing, analysis, visualization, and an interactive Shiny application.

____________________________________________________________________________________________________________________________________________
____________________________________________________________________________________________________________________________________________
**Data Sources**

Fortune 1000 Companies 2024 Dataset: https://www.kaggle.com/datasets/jeannicolasduval/2024-fortune-1000-companies

Public Company ESG Ratings Dataset: https://www.kaggle.com/datasets/alistairking/public-company-esg-ratings-dataset

____________________________________________________________________________________________________________________________________________
____________________________________________________________________________________________________________________________________________


**The files zipped**:

- Scripts_Rmardown Folder: Contains all the R scripts generated during the project, including data cleaning, analysis, and visualization code and the R Markdown file.

- APP_Rev_Rank Folder: This is the folder to ShinyApp developed for interactive exploration of revenue and ranking data.


- Data Folder: Contains the first and the second datasets.

- Images: Contains the logo and the images that we incorporated in our R markdown.

- Output.html: Is the output saved as singe .html file
